# Rywave/landing-page

## Environment

```:
node: >=14.0.0
npm: >=8.0.0
```

## Getting Started

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build
$ yarn start
```

## Running Tests and Linters
```bash
# run linters
$ yarn lint

# run tests
$ yarn test:watch
$ yarn test
```
